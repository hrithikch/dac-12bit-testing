# Tools Directory

This directory contains development and maintenance tools for the PLL sweep automation project.

## Purpose

Add development tools such as:
- Code formatting and linting tools
- Data visualization scripts
- Performance analysis tools
- Database migration scripts
- Development environment setup tools

## vs. Scripts Directory

- **`tools/`** - Development and maintenance tools for project contributors
- **`scripts/`** - User-facing utility scripts for end users

Tools here are typically used during development, testing, or system administration rather than regular operation.
